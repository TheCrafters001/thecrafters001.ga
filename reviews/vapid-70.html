Vapid

Rating out of 10: 10/10

I love this program. It allows me the convenience of a CMS, but also the complexity of HTML! Just being able to easily code a website using Vapid is amazing! Like, the code for having partial website parts that can be easily added is so cool! And all it is, is a little manipulation of HTML. All I need for the header for my website is this:

{{part part/header}}

And that is why if you look at the raw code to my website it is so messy, but I am not the one generating it, so I think that is fine. What I love is that if I want it formatted the way I want it, I can format it in any way I want. Like, here is a snippet of code I actually use for my website!

{{> part/header}}
<div class="uk-container">
    <h1>Blog</h1>
    <table class="uk-table uk-table-justify uk-table-divider">
        <thead>
            <tr>
                <th class="uk-width-small">Post Title</th>
                <th class="uk-width-small">Date Created</th>
                <th>Author</th>
                <th>Software Summary</th>
                <th>Read More</th>
            </tr>
        </thead>
        <tbody>
    {{#section posts label="Blog Posts" multiple=true}}
        <tr>
            <td>{{title}}</td>
            <td>{{_created_at format="%m/%d/%Y %l:%M %p"}} EST</td>
            <td>{{postAuthor label="Author"}}</td>
            <td>{{summary type=html editor=markdown long=true required=true}}</td>
            <td><a href="{{_permalink}}" class="uk-button uk-button-primary" type="button">Read More</a></td>
        </tr>
    {{/section}}
    </tbody>
    </table>
</div>
{{> part/footer}}

It is really simple, and I love it! I am using it to generate this website right now! The one you are reading this on. Like it?
