Hexo

Rating out of 10: 7/10

This program is okay for the most part. It can get the job done, if all you want is a simple front end blog, and making pages is a lot harder using it, but it can do.
It isn't the best, being that most of what you read for the plugins and templates is in chinese, but it works.
Now, the program itself isn't chinese, but if you want a blog with a nice template, you might want to have a tab of Google Translate open. Overall, it is okay.
Bonus is that it can export to GitHub, so that is neat.
