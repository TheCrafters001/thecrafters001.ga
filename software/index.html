TF2 Mod Installer

Windows 7 SP1 or Higher

Hi! Want to install TF2 Mods, but don't want to have to go through a pain staking process of trying to figure out how to put them in TF2? This is the tool for you! Select the mods you want, and then you just click install! Simple! I didn't make any of these mods. Please support the original creators of the mods. DO NOT INSTALL MULTIPLE HUDS AT THE SAME TIME! If you see that a viewmodel has both a Far and a Close varient, note that Far means 70+ FOV, and close means 54 FOV, and should not be installed at the same time. The following viewmodels cannot be installed because I either cannot extract them or cannot find them: Medic, Spy, Soldier. All mods in the TF2 Mod Installer are not mine. If you are the owner of a mod, and would like it removed, please contact me.

Download: GitHub | Game Jolt
