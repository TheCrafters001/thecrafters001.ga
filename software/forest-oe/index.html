Forest-OE

Forest-OE is an operating environment built on VisualBasic on the .NET Framework! I plan to make this the first ever OS to interact with Windows in a big way! Forest-OE is not a shell replacement for Windows, it is an application. It helps the user by making the Windows experience better. This is an open source project using GNU GPL V3.0!
