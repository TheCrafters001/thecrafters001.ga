Game Updater

Solo Program

GameJolt, Itch.IO, GitHub

Windows 7 SP1 or Higher

Game Updater is a free utility so you don't have to go through GameJolt's and Itch.IO's command line interfaces.

Feature List:

  • Supports both GameJolt GJPush & Itch.IO Butler!
  • Custom DLLs for easy management in the backend, and to reduce file size.
  • Lightweight
  • Free
  • Automatic Updates (kinda, you have to check manually.)

Game Updater is supported on Windows 7/8/8.1/10. All Vista support has been dropped. .NET Framework 4.7.2 or higher is required.

TF2 Mod Installer

Solo Program

GitHub, GameJolt

Windows 7 SP1 or higher

Hi!

Want to install TF2 Mods, but don't want to have to go through a pain staking process of trying to figure out how to put them in TF2? This is the tool for you!


Select the mods you want, and then you just click install! Simple!


I didn't make any of these mods. Please support the original creators of the mods. DO NOT INSTALL MULTIPLE HUDS AT THE SAME TIME!


If you see that a viewmodel has both a Far and a Close varient, note that Far means 70+ FOV, and close means 54 FOV, and should not be installed at the same time.


The following viewmodels cannot be installed because I either cannot extract them or cannot find them: Medic, Spy, Soldier


All mods in the TF2 Mod Installer are not mine. If you are the owner of a mod, and would like it removed, please contact me.
