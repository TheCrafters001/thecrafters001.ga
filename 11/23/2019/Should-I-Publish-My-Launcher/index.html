Should I Publish My Launcher?

11-23-2019
TheCrafters001
Launcher / Poll

Hello!

If you have been looking at my post for a while, you may remember that I posted that I am working on my own software launcher. As of this post, I have published beta 0.1.6, and I was wondering if I should post it on other platforms, other than GitHub. In it’s current state, it has the following programs:

  • Visual Studio
  • Visual Studio Code
  • VideoLAN VLC
  • Easy Butler (my program)

And it also includes Auto Update. To check for an update, you click “About,” and “Check for Update.” Let me know if you guys want to see this be published on GameJolt and Itch.IO!
